Unidades de medidas - REM

O rem é relativo a 2x o valor padrão de tamanho definida pelo navegador

Por exempo, os navegadores por padrão definem as fontes com 16px, usando 1.5rem conseguimos atingir os mesmos 24px

A vantagem em usar o rem é pela questão de assessibilidade, já que se a pessoa configura o navegador para usar o tamanhho de fonte maior, o rem irá respeitar, coisa que o px não fará
